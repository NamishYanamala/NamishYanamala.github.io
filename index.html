Hey, I'm Namish — a Computer Science student at UNC Charlotte concentrating in Cybersecurity, based in Charlotte, NC. I got tired of just using security tools and wanted to understand (and build) the machinery behind them.

I'm currently an Enterprise Tech & Automation Intern at Ally Financial, where I work on zero-trust security hardening — things like locking down a credential proxy so sensitive auth headers are blocked by default, filtering unauthorized headers before they reach backend services, and extending an AWS DevSecOps pipeline to enforce Terraform compliance on hybrid cloud (Nutanix) deployments.

Outside of work, I'm obsessed with the overlap between offensive security and AI agents: what happens when you give an LLM tools, a scope, and a target? That question is what's driving the project below — building systems that think like an attacker so defenders can stay ahead of one.
